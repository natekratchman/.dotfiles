# Example Skill Invocation Prompts

Templates demonstrating how users might invoke Ruby-based Skills.

## Creation Prompts

```
Create a new skill for processing financial reports with validation
```

```
Build me a skill that handles customer support tickets with priority routing
```

```
Generate a Ruby skill for code review with different strictness levels
```

## Modification Prompts

```
Update my existing skill to add a new workflow branch for urgent requests
```

```
Add error handling to my skill's processing pipeline
```

```
Extend my skill with support for batch processing
```

## Complex Prompts

```
I need a skill that:
- Processes uploaded documents
- Routes to different handlers based on file type
- Has configurable output formats
- Validates inputs before processing
- Logs operations when in debug mode
```

```
Create a multi-stage skill with:
1. Input gathering phase
2. Validation checkpoint
3. Processing with retry logic
4. Output formatting options
Use Ruby constants for all configurable values
```

## Minimal Prompts

```
Simple skill that adds timestamps to documents
```

```
Quick skill for formatting code snippets
```
